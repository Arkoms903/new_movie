## 🌟 Overview  

A **Content-Based Movie Recommendation System** built using **Python, Pandas, Scikit-Learn, and Streamlit**.  

It recommends movies similar to your selected movie based on **genres, cast, crew, keywords, and overview** using **Cosine Similarity**.

✨ Includes movie posters using **OMDB API**.
